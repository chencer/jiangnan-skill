# 负面清单 anti_patterns.md

## A. 通用 AI 腔病灶

1. **短句无脑堆砌** — 连续 4 个以上 ≤8 字的句子，且无节奏意图（即不是为了喘息/停顿，只是句子切得碎）
2. **比喻词高频堆叠** — 同一段落出现 2 次以上仿佛/宛如/像是/如同/似乎
3. **段尾强行升华** — 每段结尾用议论/抒情句总结情绪，例如"这就是……""他意识到……""命运在此刻……"
4. **排比滥用** — 一段内超过 1 组三连排比（"他××，他××，他××"）
5. **形容词轰炸** — 同一名词前挂 2 个以上修饰词（"漆黑阴冷的""深沉压抑的沉默"）
6. **空洞哲理句** — 直白总结型句子，如"这就是青春""人总是如此""命运不过如此"
7. **对话标签单调** — 全篇对话标签只有"XX说"，或反向滥用花式标签（"他低吼道/轻叹道/淡淡道"）；正确做法是"说"与动作描写交替
8. **欲言又止滥用** — "想说什么却没说/翻出来看了一眼又没说"类句式，全篇最多 1 次，且必须搭配具体信息点（读者能推断出他到底想说什么）；否则视为无信息量的空转，直接删除（第1轮反馈新增）
9. **单段感叹号堆积** — 每段 ≤1 个感叹号，全篇平均值达标不能掩盖局部堆积（第1轮反馈新增）
10. **空话搪塞设定** — 禁止用"你知道的/不重要/你清楚的"之类空话代替具体设定信息；核心冲突（谁、要什么、为什么）必须至少有一处具体锚点（第1轮反馈新增）

## B. 量化红线（基于 Phase 0 实测，baseline_longzu.json）

| 指标 | 允许区间 / 上限 | 基准值 | 一票否决？ |
|---|---|---|---|
| 平均句长 | 20-30 字 | 24.9 字 | 否 |
| 短句占比（≤8字） | **≤35%** | 29.7% | **是（超过即重写）** |
| 各句长分布 | 17%-35% 每档 | ~30/22/27/21% | 否 |
| 平均段落长 | ≥40 字 | 66.8 字 | 否 |
| 比喻词合计（仿佛/宛如/像是/似乎/如同） | ≤20次/万字 | 13.09次/万字 | 否 |
| 感叹号频率 | 25-62次/万字 | 41.29次/万字 | 否 |
| 省略号频率 | ≤50次/万字 | 40.2次/万字 | 否 |
| 对话字数占比（高压场景） | 15%-30% | 战斗章 23.2% | 否 |

**短篇样本（1500-3000字）折算说明：** 频率类指标按每万字换算对照，允许 ±30% 额外浮动，但短句占比红线（35%）和粘连句红线不放宽。

## C. 分句级红线（防逗号粘连作弊，第2轮反馈新增）

江南基准值（由 style_stats.py 对 corpus/longzu.txt 实测，2026-07-09）：
- 平均分句长（逗号切）：**10.3 字**
- 极短分句占比（≤5字）：**19.4%**
- 粘连句占比（≥4分句且均长≤7字）：**1.4%**

红线阈值：
| 指标 | 红线 | 基准值 | 一票否决？ |
|---|---|---|---|
| 平均分句长（逗号切） | **≥ 8.2 字**（基准×80%） | 10.3 字 | 否 |
| 极短分句占比（≤5字） | **≤ 25.2%**（基准×1.3） | 19.4% | 否 |
| **粘连句占比（≥4分句且均长≤7字）** | **≤ 1.8%**（基准×1.3） | 1.4% | **是（超标即重写）** |

**粘连句红线不随样本量放宽。** 句号级指标全绿但粘连句超标，同样判定碎片化。

**逗号粘连的根因说明：** 把内容塞进逗号串联的短语碎片，让统计看起来是长句，实际是把"他，站，等，看，想，说"这种动词碎片用逗号连起来的假长句。正确做法：每一句都有完整的主谓，改写时回到要表达的意思，用一个新句子重说。

## 检查流程

写完逐句过以下清单，发现任意一条触发即停下重写：

- [ ] 有没有连续 4 句以上短句（≤8字）却无喘息用意？
- [ ] 同段出现 2 次以上比喻信号词？
- [ ] 任何一段结尾在抒情/总结/升华？
- [ ] 有没有"这就是""他意识到""命运"之类的哲理词？
- [ ] 对话标签是否全部是"XX说"或全部是花式动词？
- [ ] "欲言又止"句式超过 1 次？或出现了但没有具体信息点？
- [ ] 某一段感叹号超过 1 个？
- [ ] 有没有"你知道的/不重要/你清楚的"空话代替具体设定？
- [ ] style_stats.py 短句占比是否超过 35%？（机器检查）
- [ ] style_stats.py 粘连句占比是否超过 1.8%？（机器检查，一票否决）
- [ ] n-gram 命中数是否为 0？（机器检查）
