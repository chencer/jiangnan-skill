## 基本信息
- 场景类型：世界观交代（龙的本质定性 + 拉斯普京考察团血腥往事 + 龙的死因揭秘）
- 在场人物原型：邦达列夫（外来见证者/贵族继承人）、博士（守秘者/信息投喂方）
- 核心目的（一句话）：借冰下龙骸把"第四域"世界观与拉斯普京百年悬案一次性砸给读者，同时完成博士与邦达列夫的利益结盟。

## 火力分配
- 重墨对象：
  1. 冰下龙骸的外形与压迫感（骨刺、鳞片、大理石般的眼、铁面般的脸）——静态定格，反复雕琢。
  2. 茨冈人捕鱼遇龙的目击场面——动态特写，冰洞爆水+金色火眼+集体致疯。
  3. 变异动物素描一幕（蛇熊配种）——单幅画面被抠到血淋淋的分辨率。
- 简笔内容及带过手法：
  - "第四域-δ计划"的科学解释：靠一句"进化之树"打比方直接跳过，不铺演推导。
  - 拉斯普京从莫斯科到北冰洋的行程：一句话行军，牧首、狗拉雪橇、猎犬引路都是名词点到。
  - 戈利奇纳家族几十年挖掘：整段压缩成"先找到骨骸-再找到岩壁-最后找到龙"三拍。
- 支线位置：入口——邦达列夫祖辈与拉斯普京的私交（勾邦达列夫家族线）；出口——戈利奇纳上将暴卒、经费断链（勾出接下来的资助关系换手）。

## 分镜序列
1. 大远景 / 特写切换：冰层里苍青色骨骸的全身长度 → 眼睛的大理石光泽（有人看你的错觉）。
2. 对切镜：博士与邦达列夫隔冰对话，博士手指点向脊椎中段的黑刺。
3. 微距：刺刀被辨认——李氏长步枪、陨铁——一件小物锁死"死亡时间不超过百年"。
4. 时间闪回（1908冬）：茨冈人凿冰捕鱼 → 无鱼 → 老者预警 → 冰洞爆水 → 巨颅探出 → 金瞳吼叫 → 近者疯、远者存。
5. 血渗冰面的中景：手指触血 → 颅骨变形 / 生鳞 / 肩生第二头 → 剪到邦达列夫想起北极熊双脊椎（前情回勾）。
6. 拉斯普京线：狗拉雪橇进冰原 → 洞口 → 岩壁裂缝 → 龙巢 → 双面话术分别哄骗牧师与科学派。
7. 素描特写（画中画）：博士递纸卷 → 邦达列夫翻开 → 蛇熊配种画面被从纸面上拽出来当实景写。
8. 收束镜：封洞、假地图、拉斯普京独走 → 拉回冰室，博士给出戈利奇纳暴卒的现况。
- 时间伸缩点：
  - 冰下静止的"现在时"→ 一句"1908年的冬天"急速倒带80余年。
  - 变异实验一整段用列举法把几十天压成三行（熊/蛇/猫）。
  - 从拉斯普京离开到红军将军发现笔记：一句话跨过大半个世纪。
  - 素描内的动作反而被慢放到肌肉级别（蛇缠、尾刺穿腹、吞头、反咬）。

## 技法清单
- 比喻：
  - "大理石般的光泽"——视觉，把死物拟出被凝视的错觉。
  - "进化之树"——概念可视化，把生物学抽象降格为图像。
  - "脸上像罩着铁面"——视觉+触觉联觉，为后文对照龙骸铺锚。
  - "血红色液体的秘药"——中世纪炼金意象，给龙血提前贴价签。
- 感官传导链：有。视觉（苍青骨骸）→ 错觉（龙在看你）→ 听觉（震耳吼叫）→ 神经（致疯）→ 触觉（手指触热血）→ 身体（生鳞、长头）。一条链把"看"推到"变形"。
- 对话：
  - 密度：中偏高，博士长段独白+邦达列夫短句冷插。
  - 标签风格：极简，"博士说""邦达列夫说"，几乎不加副词。
  - 潜台词错位：博士在讲神话与灾难，邦达列夫回一句"这淫棍真是一朵奇葩""只要那淫棍不是想向曾祖母求爱"——把史诗话题一巴掌拍回八卦层，典型幽默模式的降格。
- 金句：
  - 触发源：博士被邦达列夫"有价值的人在任何时代都会被尊重"这句白天旧话打动。
  - 落点后接什么：博士补一句"只有懦夫才被时代束缚，有能力的人创造时代"——紧跟着不是煽情，而是邦达列夫举杯敬"真理"并立刻转问"这条龙现在是死了么？"。金句落地即被酒杯截断，回到冷叙事，不允许情绪停留。

## 可复用度
- 评级：A
- 一句话点评：教科书级"文物+独白+闪回+素描"四层套娃，重物件（龙骸/刺刀/素描）承担世界观，重人物一句冷话就把史诗降格，是江南做长篇设定倾泻时的标准打法。
