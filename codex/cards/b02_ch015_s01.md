# b02_ch015_s01 · 卑贱藏狮·摄政王的一句评语

## 基本信息

- **卷/章**：卷二 · 第八章《极乐天都 / Pure Land》
- **锚点**：ch_015_第八章.txt L6-L7
- **场景类型**：A级 · 金句收尾（章首孤悬式评语，人物首现的目光定格）
- **模式**：抒情模式冷调（第三人称摄政王视点，无环境无动作）
- **人物**：摄政王（说话者，尚未具名）、樱井小暮（听话者，被三字带过）、照片中的路明非（被评价者）

## 火力分配

- 一句台词承担全部戏剧任务：否定→重指→评价三步走
- "含泪凝睇"四字暗埋歌舞伎《杨贵妃》的余韵，为下文 L80 男人扮女形埋线
- 三个矛盾形容词并列成排（卑贱 / 悲伤 / 藏着狮子），最后一个升格为动物意象（M03）
- 无任何环境交代，读者不知说话者是谁、在哪、对谁说——章首意象完全孤悬

## 分镜序列

1. 章名双语并置：极乐天都 / Pure Land（净土的反讽预告）
2. 空行
3. 摄政王一句独白，先否定读者预期（不是加图索继承人）
4. 重新指认对象：那个照片角落最不起眼的男孩
5. 三段递进评价：卑贱—悲伤—藏着狮子
6. 场景在这句话上戛然而止，跳到极乐馆全景

## 技法清单

- **章首末同句闭环（T01）**：这句独白与 L110 一字不差复现，构成整章的口衔尾环。读者第一次读只当是一句形容语，读到章末看过赌场血案、樱井小暮的驯服、歌舞伎表演后再遇同句，才理解"藏着狮子"这四个字是宣战书而非赞美——它把整章所有的猎人心态、试验品、亲吻中的催眠都收束进对路明非的一次锁定。
- **反差命名·以X定义X（M04）**："卑贱"和"狮子"在同一个短语里对撞。江南不写"看似卑贱实则强大"，而是把两个矛盾的评价直接叠放，用"却又藏着"三字做扣，让读者自己去调和这两个词的张力。
- **比喻·物种升格（M03）**：把一个中国胖子少年直接抬到狮子层级，且用"藏着"而非"像"——不是形似，是内里。这是江南标志的越界命名，与《龙族》里楚子航被称作"暗夜里的君王"同构。
- **章首孤悬·信息不足（N02）**：说话者的身份、地点、语调、听众全部悬空，只有一句台词。读者要读到 L80 才知这男人是歌舞伎女形，读到 L111 才知他叫将军。江南用信息饥饿感牵引读者往下走，一句话就是一根鱼钩。
- **视线做接触**：全句的动作动词只有"盯着"和"目不转睛"，是视觉的锁定动作。摄政王没有起身、没有伸手，只是看——但这一看是狩猎，从这一秒开始路明非已经进入猛鬼众的猎场，虽然远在东京他自己不知。这是 N04 伪装破绽·读者先于角色知情·延迟引爆的顶级示范。
- **含泪凝睇·未爆预告**：形容摄政王的这四个字看似写他还沉浸在《杨贵妃》里，实则预告了他后文的女形身份和精分气质。首读时读者会当作性情形容词，第二遍才知是身份泄露——江南把身份藏在情态里。

## 可复用度（A）

- **适用场景**：反派/大 boss 首次登场且不肯露脸；一个能左右主角命运的评价者远远做出锁定判决；章首孤悬式题引，需要一句话让读者对整章的走向产生不安；对主角进行"看似贬低实则升格"的物种级评价。
- **迁移公式**：
  1. 章名给一个反讽意象（"净土/极乐"用来包裹后文的血腥与堕落）。
  2. 一句台词，先用否定句排除读者以为的重点人物（"不是A，而是B"）。
  3. 用三段递进形容锁定评价对象，前两段用负面词（卑贱/悲伤/怯懦），最后一段跳跃到动物或神性升格（狮子/幽灵/古神）。
  4. 说话者的身份不给、场景不给、听众不给——只有台词和一个含糊的情态（含泪凝睇 / 半合眼 / 手指抚过刀鞘）。
  5. 章末在完全不同的语境下把这句话一字不差再放一次，让整章的所有情节回头做注脚。
- **陷阱提示**：三段递进如果全部同向就成了排比作文；关键在于前两段是"看起来"，最后一段必须是"其实"的物种越界。省掉"却又藏着"这类扣词就会失去张力。另外含糊情态词一定要选后文能兑现的（歌舞伎伏笔 → 含泪凝睇），如果只是氛围形容词就浪费了预告位。
