## 分镜序列

1. **诱供**：伪装克格勃军官，让父母亲口把女儿标价出售
2. **加码**：从"接回团聚"vs"再次捐献"给出二选一，答案早已锁死
3. **递刀**：十万卢布现钞十扎推过桌面，父亲搓着手要求抹去父名
4. **爆点前静音**：父亲伸手抓钱，"但他没能说完这句话"——半句被军刺打断
5. **屠戮慢镜**：黑色军刺穿心 → 血槽迸射 → 反手刺入夫人 → 缓慢拧动 → 血喷涌 → 抖腕抽刃 → 天鹅之死姿势倒下
6. **自我合理化**：擦拭刀身，独白"蝼蚁"，编好给女孩的谎言
7. **收尾三动作**：踢钞票 / 放卷闸门 / 掰断钥匙在锁孔 → 转身扎进风雪

## 技法清单

### ① 伪装身份的语言破绽（延迟引爆）
- 少年一路用军官口吻走完整套官方流程，直到一句混入的奇怪贞操比喻漏出伪装破绽（见 E-b01ch007s03 第1条）
- 江南不解释这是谁，用旁白式破绽埋设句点明父母沉浸于贪念而未察觉（见 E-b01ch007s03 第1条）——**破绽当场埋下，读者比角色先一步凉透**
- 复用要点：反派卧底时，让一句突兀的比喻先于身份揭露，读者的后颈发凉早于角色

### ② 父母的自愿献祭 = 免死金牌的反向使用
- 越激动地卖女儿，越坐实"该死"
- 关键台词全是以感叹号结尾的主动献祭式表态（见 E-b01ch007s03 第2条）
- **让角色自己宣判自己**，少年只是执行者

### ③ 军刺屠戮的"慢动作+器械学"
- 不写"惨叫""恐惧"，写**器物参数**：黑色军刺 / 血槽 / 反手刺入 / 缓慢拧动 / 抖腕抽刃
- 血是从血槽两侧"喷涌"和"迸射"，用**机械原理**替代情绪渲染
- 收尾用芭蕾术语式的优美倒地姿势（见 E-b01ch007s03 第3条）平衡血腥——极暴力配极优雅

### ④ "但他没能说完这句话"——半句截断法
- 父亲的话被军刺打断在一个逗号之后
- 江南不写"话音未落"这种俗套，写"没能说完"，五个字杀掉一个人
- 复用要点：想让一个动作/一句话被暴力截断，用"但他没能……"起一段

### ⑤ 屠戮后的私人逻辑（幽默模式的孤独）
- 少年拿出杀人的理由不是"正义"也不是"愤怒"，而是对女孩的一句承诺（见 E-b01ch007s03 第5条）
- **降格宏大**：谋杀 → 帮朋友完成一个小承诺
- 用"难堪"这种社交性词描述屠杀失败可能带来的心情（见 E-b01ch007s03 第5条）——落差即张力

### ⑥ 编谎言的温柔（最冷的一刀）
- 他要骗女孩：父母终身等候却未能撑过严冬的温情谎言版本（见 E-b01ch007s03 第6条）
- **谎言的措辞比真相更文学**，用抒情句式包装虚构，读者在这里心里最冷
- 复用要点：让反派用美好句式撒谎，比让他说实话更狠

### ⑦ 收尾三动作 + 一个逆行
- 踢钞票到死者手中（构图讽刺）/ 放下卷闸门 / 掰断钥匙在锁孔（不留后路）
- 最后背影逆行扎进风雪的收尾动作（见 E-b01ch007s03 第3条）——**背影+风雪+不回头**，江南经典逆行收尾
- 期间他哼着歌，神情与行为分裂的细节（见 E-b01ch007s03 第4条）——表情与行为的分裂

### ⑧ 平静后的自我否认逻辑
- 第一层否认（见 E-b01ch007s03 第5条）——先声明不在乎
- 第二层否认（见 E-b01ch007s03 第6条）——再声明不屑
- "但我答应过那个傻妞"——最后才承认真正的动机
- **三层否认递进，把真心话藏在第三层**，符合少年那种既高傲又别扭的人设

## 可复用度（A）

**A级理由**：
- 屠戮场面的"器械学+慢镜头+无情绪词"是江南暴力美学的标配写法
- "但他没能说完这句话"这种半句截断法可以直接套模板
- 反派用美好句式编谎言的技法在任何黑暗人物身上都能复用
- 三层自我否认逻辑（不在乎→不屑→其实是为了……）适合所有"嘴硬心软"角色
- 收尾三动作+背影+风雪，是万能的冷抒情收束

**移植场景清单**：
- 任何伪装身份的诱供-爆发戏
- 反派的"温柔杀"戏码
- 角色需要在暴力后立刻回到人味的场面
- 需要让读者比角色先感到不寒而栗的破绽埋设

**注意事项**：
- 屠戮段的"优美姿势"必须真的优美，配不上极端暴力就成了猎奇
- 三层否认不要写成四层五层，会啰嗦
- "蝼蚁"这种词一段里只能用一次
