# b02_ch018_s13 — 太子揭秘·龙渊确认·黑卡成交

## 基本信息
- 书目：龙族 II（悼亡者之瞳）第十一章
- 行号：L210-L225
- 场景类别：高压冲突 A
- 一句话概括：施耐德把"太子"这个隐身盟友的存在交出去，用一段几乎冷得像技术手册的独白解释龙渊为什么必须下潜；曼施坦因当场把黑卡推过桌面，双方在最后一句"那么成交"里达成共谋。

## 火力分配
- 世界观揭示（45%）：太子这个 ID 的历史、克格勃档案、列宁号沉船的来龙去脉
- 政治冲突（35%）：校董会 vs 秘党的路线之争、曼施坦因作为文职人员的"噬罪者"选择
- 情感余温（20%）：从 s12 累积的父子情作抵押，让此处的信任移交有分量

## 分镜序列
1. L210｜秘密掉落｜"太子，是太子"——不解释，让名字先落，再往回补
2. L211-L212｜档案语气讲传奇｜太子 ID 消失又复活、公布克格勃文档，全程用"我们怀疑""据说""大概"——把史诗信息处理成情报简报
3. L213-L214｜自问自答的伦理｜曼："如果这是另一个诱饵，你们为什么还要去咬钩呢？" 施："只要我们确认那是一枚胚胎，我们就不能任它孵化。" 一问一答用最冷的语法讲最狠的选择
4. L215-L216｜盟友身份定性｜"某种程度上来说，他是我们隐藏在暗处的盟友，虽然他从来不出面作战"——一句话给太子盖章
5. L217｜技术道具亮相｜大屏幕、进度条、心跳信号、32%绿色安全——用技术仪表把宏大生死量化为一个百分比
6. L218-L219｜人物性格顺手一刀｜施耐德对装备部"那帮神经病"的一句吐槽，把技术信任和不信任同时表明
7. L220｜道具揭幕反转｜曼施坦因抽出黑卡推过去——L1-L219 全部铺垫此一动作
8. L221-L222｜自认罪名｜"这个罪就由我来吃掉吧"——文职人员领罪，用一个食物动词把政治后果日常化
9. L223-L224｜真心亮相｜施耐德指出曼施坦因来的真正目的，曼直接承认"我只是要确认你确实在乎那些年轻人的命"
10. L225｜一句成交｜"那么成交。"三字截断，全场收束

## 技法清单
- **N02 宏大场面轻描淡写**：L212 克格勃档案、苏联解体、北方舰队军舰——世界级历史像天气预报一样被念完
- **P01 道具揭幕反转**：L220 黑卡抽出的一瞬间，L179 起铺垫的整段"曼施坦因是校董会的人还是施耐德的人"的悬念被翻盘
- **N06 占有/情感宣告·塞进日常腔**：L222 "这个罪就由我来吃掉吧"——把政治承担说成一顿饭
- **H03 日常语汇收束宏大/神圣时刻**：L217 龙类胚胎的孵化率用"进度条""绿色安全""32%"这些游戏化仪表处理
- **N04 伪装破绽·读者先于角色知情·延迟引爆**：L223 施耐德一句"其实你来这里的目的就是想知道这件事的内幕吧"把 s12-s13 整段家史重新解释为一场精密的软磨——读者回头才发现曼施坦因讲家史时就在下套
- **N01 冷短句截断·不接情绪**：L225 "那么成交。"四字段落收尾，不给握手不给沉默
- **M03 比喻·物种/神性升格**：L214 胚胎"随时可以破壳而出""结茧""进入成年期"——用昆虫谱系描述古龙
- **N07 归因反转·降格自救**：L222 曼施坦因自嘲"我的血统和能力都很一般，跟我的变态老爹完全没法比。但是当个噬罪者的话还够格吧"——用降级承认换取实际主动权
- **H01 想/说落差**：曼施坦因心里想拿到证据保住少年们的命，嘴上说的是"这个罪我会想办法吞下去"，把救人这件事讲成消化事故
- **T02 倒计时锁定结构**：L214 "越往后胚胎的孵化率越高，一旦它拥有自我意识就来不及了"——用时间递减制造压迫
- **P02 道具无言情感承载**：黑卡从卡槽抽出、推过桌面、被接受——三个物理动作替代整段情感承诺

## 可复用度（A）
- 结构可复用：**"高压谈判 = 一方交出核心秘密 + 另一方推出核心道具 + 一句短句成交"**的三拍收束模型，可移植到任何双方博弈场景
- 语调可复用：**技术简报腔讲传奇**（L212 克格勃段、L217 心跳信号段）是江南写世界观展开的常态语法
- 单一细节警告：**"太子" ID + 克格勃 + 列宁号沉船**是龙族独有神话谱系，仅作结构参考
- 单一细节警告：**"噬罪者""吃掉这个罪"**是曼施坦因的专属修辞习惯，移植时需换更中性的动词
- 反抒情样板：L225 "那么成交"作为一场情感重头戏的收尾——**最有分量的时刻用交易术语**，是江南反抒情抒情法的高频写法
