# b02_ch022_s01 · 核动力舱故障 · 决策危机

## 基本信息
- 来源：ch_022 第十五章 潜龙升空之海
- 指针：L5–L45
- 场景类型：高压冲突 / 多方决策危机
- 等级：A
- 一句话概括：投掷核动力舱之后设备失灵，深海八千米下的三人被通知必须自愿去死一次；恺撒替队伍下潜，把牺牲说成骄傲。

## 火力分配
- 主打技法：H01 想/说落差 · N06 占有/情感宣告·塞进日常腔 · N03 神性/巨物降格·日常语汇化
- 副线技法：T02 倒计时锁定结构 · N01 冷短句截断·不接情绪
- 情绪曲线：设备失灵（错愕）→ 通讯博弈（怒吼）→ 恺撒接管（冷）→ 骄傲宣言（烈中带笑）→ 阳光比喻（升温收束）

## 分镜序列
1. **L9–L11**：数据播报开场——用中子密度、当量、生还几率把危险量化成技术语言，制造冷压迫（T02）。
2. **L13–L16**：一声"等一等"晚了半拍——楚子航呼喊、挂钩已开、核动力舱下沉，动作先于警告，读者提前几秒感到坏事已成（N04的雏形）。
3. **L17**：路明非用"装备部耍我们么"这种小卖部式吐槽收束百万吨级危机——把国家级装备说成外卖单（H03 / N03）。
4. **L26**：源稚生一拳砸栏杆，怒吼里夹着自责——身体动作代替"我很崩溃"（N05）。
5. **L29–L37**：无线电通讯战——源稚生放弃伪装说实话，恺撒用"平塔岛象龟""婚礼"等私人清单把死亡砍价成生活账单（M01降格）。
6. **L38–L44**：恺撒摘耳机、卡脖子、按肩膀——三段身体动作全部替代情感宣告，队长身份的宣布是通过物理姿势完成的（P02 / N05）。
7. **L44**：耻辱清单式的骄傲宣言——把"我为你们去死"翻译成"我不愿意在我的人生里出现你们中的某个死了而我活下来的段落，那样我讲不出口"（N06）。
8. **L45–L46**：楚子航一句"你真是一辈子只为骄傲活着的人啊"落地，紧接一句"在海蓝色的眼瞳中看见刺眼的阳光"——冷句 + 意象收束（N08）。

## 技法清单
- **H01 想/说落差**：源稚生心里的绝望（一个小小疏漏，几乎没有逆转的机会 L22）与他嘴上通讯频道里的"这是我们唯一的机会"（L29）之间落差极大；恺撒心里怕死，嘴上是耻辱、骄傲、婚礼。
- **N06 占有/情感宣告·塞进日常腔**：恺撒不说"我要救你们"，说"我的命比你们值钱"（L44），把牺牲翻译成商业估值。反抒情的抒情。
- **N03 神性/巨物降格·日常语汇化**：百万吨当量的核爆威力被路明非以"装备部耍我们么"（L17）一句降格；恺撒把死亡比作"再也去不了天体海滩卖防晒油"（L35）。
- **T02 倒计时锁定结构**：中子密度120%、20分钟、100万吨、8分钟氧气、5分钟深海行走——数字层层递减制造压迫。
- **N01 冷短句截断·不接情绪**：L45楚子航一句"你真是一辈子只为骄傲活着的人啊"之后没有心理描写，直接跳到L46的视觉画面。
- **M03 比喻·物种/神性升格**：结尾"海蓝色的眼瞳中看见刺眼的阳光"——把深海冷光升格为热带阳光，把恺撒瞳孔升格为一个日出的容器。
- **P02 道具无言情感承载**：耳机被摘、被扔、被戴的三段动作（L38 / L83），道具在三人之间迁移即是队长权柄的传递。

## 可复用度（A）
- **可仿写场景**：任何"上级要求下级去执行必死任务，且必须由下级自愿承担"的高压桥段——航天故障、拆弹现场、火线撤退、手术台上的死亡分工。
- **仿写要点**：
  1. 危机不要直接说"完蛋了"，用数据、百分比、剩余时间去替代——冷才有力。
  2. 主角接过任务时不要说"我愿意"，让他说一句"我不愿意，可我还是会做"（L36）。
  3. 情感宣告一定要包一层日常外壳：婚礼、防晒油、报销单、履历表都可以。
  4. 队伍身份的转移用物件（耳机、钥匙、肩章）完成，不要用台词说"你接替我"。
- **落地钩**：
  - 数字倒计时序列 → 中子密度、氧气、氧压、氮醉时限、电池百分比等任选三条构造密度。
  - 反英雄宣言模板 → "别以为我是为了X才Y，我只是Z"。
