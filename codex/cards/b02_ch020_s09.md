## 技法清单

- N02 宏大场面轻描淡写——辉月姬用平静的电子女声报出「高天原」，把整个日本古史重构压在两个字里。
- T03 时间多段伸缩——认知重构一段极压缩（一句一层身份），废墟苏醒一段极拉伸（肺螺一层一层剥落）。
- V02 感官全链·完整序列——鸟居阴影（视）→ 呻吟（听）→ 碎石敲外壳（触）→ 恺撒抽自己嘴巴（体感）→ 头皮 / 冒汗（生理）。
- M03 比喻·物种/神性升格——把日本天皇写成「龙种」，把神家族改写成「一个龙家族的历史」。
- M01 比喻·降格（日常器物代宏大）——「圣斗士不会被同一招击败两次」以及「打自己的脸觉得痛，不像在做梦」，用漫画梗和自扇耳光处理神迹级恐惧。
- H01 想/说落差——路明非心里已经猜到答案，嘴上先甩一句「妈的难道天皇家族真是龙种？我上历史课还真相信帝王将相宁有种乎呢！」
- H03 日常语汇收束宏大/神圣时刻——「尼玛我们的日本分部是诸神后代？」用最市井的一句，收住整段神话重构。
- N01 冷短句截断·不接情绪——「废墟开始摇晃，这不是海底地震还能是什么？」不铺情绪就切到下一段指令。
- N04 伪装破绽·读者先于角色知情·延迟引爆——「鸟居就是一种门」的破绽在 s08 结尾已经埋下，此处配合辉月姬三字揭晓，读者比恺撒早半步凉透。
- P01 道具揭幕反转——声纳屏幕从「一个巨大的心跳」翻转为「无数同步的心跳」，红点即揭幕道具。
- I02 射灯揭幕·庞然大物首现——迪里雅斯特号穿过鸟居进入阴影，是一次反向的射灯揭幕（灯灭而非灯亮）。

## 可复用度（A）

- 强复用：三段式电子播报（找到匹配 → 顿一下 → 报关键词），任何 AI/系统揭晓真相的场景都能套。
- 强复用：身份重构的四层套娃（A 是 B？→ 不，A 是 C → 真正的 B 是 D → 那 E 又是什么？）。
- 强复用：「XX 就是一种 YY」的破绽引爆句式——鸟居 = 门；可迁移到任何日常物件突然变成禁忌符号。
- 强复用：假故障真苏醒——先让角色喊「是不是 X 出问题？」再由权威角色否认「不是 X」，然后揭示真相远比 X 恐怖。
- 中复用：越洋电话/通讯延迟带来的决策孤悬感（恺撒对着「平塔岛象龟」下决心）。
- 慎用：直接把某国神话重写成龙族史这种大操作，一部书用一次即可。
