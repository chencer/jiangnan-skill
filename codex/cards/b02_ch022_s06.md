# b02_ch022_s06 路鸣泽登场·交易的伦理磨损

## 基本信息
- 位置：ch_022 第十五章
- 行号：L123-L166
- 类型：日常闲笔
- 火力级别：A
- 概述：外面尸守嗜血扑舱、酒德麻衣顶舱斩杀，舱里路鸣泽穿着和服木屐坐在控制台上，用按摩师/业务员/薯条番茄酱等日常道具铺开一次交易劝诱。谈话表层是段子，暗层是"死三次"论、报复世界论、以及路明非"卖了一半生命还是一穷二白"的自我盘账。

## 火力分配
- 情感浓度：高（藏在段子腔与和服木屐里）
- 密度覆盖：段子—科普—报价—抒情—报价—抒情—金句—拒绝—吐槽，交替铺陈
- 命中层：a 想说落差 / b 降格幽默 / c 数据化情感 / d 金句落点
- A 级理由：路鸣泽台词全程用商务话术、按摩广告、旅行手册、名人名言四种日常语域轮换，情感杀伤（尸守/龙类/死三次/被遗忘）却每次都被这些语域一口吞掉；路明非的心理反抗仅靠"眼角抽动一下"、"心里悄无声息地疼痛了一下"两个极短动作完成，情绪一次不外放。

## 分镜序列
1. L123 路鸣泽首次入镜：藏蓝和服、木屐白袜、双手托腮坐在控制台上、脸蛋像红苹果——用日式温泉少女画风盖在魔鬼身上。
2. L124-L125 交代出场逻辑：搭讪失败/贵宾候车室/女魔鬼——把"跨维度救场"讲成"取消旅行"。
3. L126-L128 业务范围科普：日本不归他管但海疆以外可以，顺便把当天麻烦升级为"上千尸守 + 一条古代种纯血龙"。
4. L129 一段真正的战术情报（尸守的胎衣、脑死、嗜血属性、精神刻印），用生物学讲义腔化解。
5. L131 恺撒的昏迷被解释成"幻觉抱婚纱诺诺进洞房"——把队友的濒死翻成粉色小说。
6. L132 路明非的第一次情绪泄露：眼角抽动一下、扭头避开——两笔完成。
7. L133-L134 尸守拍窗，路鸣泽用"下贱的东西来凑什么热闹"再次降格威胁，命令外面姑娘"吓唬一下"。
8. L138-L142（空行焊接，外景）酒德麻衣鳞片张开、拔刀、双刀天羽羽斩布都御魂、旋转出漩涡——舱外一段冷血的高浓度武技单节。
9. L144 报价第一轮：谁在害你？1/4 生命。
10. L146-L148 报价第二轮：杀光尸守 + 击败古代龙类 = "东京半岛酒店三星料理艺妓大白腿"。用旅游手册话术做定价。
11. L150 路明非内心独白：反复叫停的"心底深处的呼喊"，与"穷得只剩卡贷"的自嘲并置。
12. L151 空气凝结的对峙一句。
13. L152 路鸣泽的自我辩护段：魔鬼要诱惑客户 / 我不是坏人 / 价格公道又能干——三段递进反证自己。
14. L153-L154 路明非语塞，接一段"他从小到大究竟想要什么"的自问自答，落在"最想诺诺会喜欢他"。
15. L155 路鸣泽切旅行手册腔：新宿霓虹、北海道温泉猴子、秋叶原女仆黑丝、新干线花瓣——用美景清单为"活的日本"背书。
16. L156 一句路明非的判词："死的东西有什么好？"
17. L157 死三次论（生物学死/社会性死/被遗忘死），用课本讲解腔说出。
18. L159-L161 假想葬礼清单：楚子航妈妈晕倒 / 加图索家玩命 / 你只剩叔叔婶婶胖堂弟——用宾客名单核算悲痛值；诺诺哭在恺撒墓碑上；"有的人活着他已经死了"课本引用。
19. L162 路明非心里"悄无声息地疼痛了一下"——第二次极短情绪泄露。
20. L163 报复世界的金句"宁可被人憎恨而牢记，也不要毫无存在感地被遗忘"，末尾轻描淡写地补"好像是什么名人名言来着"，自我拆解。
21. L164-L166 路明非大声拒绝 → 路鸣泽用"莎剧演员台词功底"自嘲，提议出去透气。

## 技法清单
- H01 想/说落差：路鸣泽内心是屠日本、扒婚纱的极端欲望，嘴上却是按摩广告、旅行手册、名人名言。
- H03 日常语汇收束宏大/神圣时刻：屠灭尸守 + 击败纯血龙 → 兑换成"米其林三星 / 艺妓大白腿"的商品套餐。
- N03 神性/巨物降格·日常语汇化：8000米深海说成"脚上绑石头被丢进海里的海盗"；古代种纯血龙说成"那个大东西"。
- N06 占有/情感宣告·塞进日常腔：最深的占有欲（"她是你的囚犯，住在你的牢笼中"）出现在按摩师台词的语境里。
- H09 自我辩护·递进被反噬：L152 三段式—"我是魔鬼嘛/要是每天救灾还是魔鬼么/价格公道又能干"，越辩护越坐实。
- N05 身体行为替代情感表述：路明非的抗拒全靠"眼角抽动一下"（L132）"心里悄无声息地疼痛了一下"（L162）两个极短动作。
- N08 金句落点·接日常动作截断：L163 报复世界的名人名言后立刻自嘲"好像是什么名人名言来着"；L157 死三次论后紧跟"你想说什么"。
- T04 双重视角空行并置（禁过渡句）：L137 与 L138 之间空行焊接，舱内谈判段与舱外酒德麻衣斩杀段并置，无过渡句。
- P02 道具无言情感承载：藏蓝和服、木屐白袜、控制台、番茄酱薯条、玩具沙漠之鹰（延伸到 s07）——道具承担了魔鬼的日常化伪装。
- M01 比喻·降格（日常器物代宏大）：把交易灵魂比成买健康理疗，把炸沉日本比成"海底废墟没意思"，把死亡比成断气/下葬/被忘记的三段流程。
- N04 伪装破绽·读者先于角色知情·延迟引爆：读者早知道路鸣泽是魔鬼，路明非却仍在段子里被一点点拉近悬崖边缘。

## 可复用度（A）
- 高复用：让反派用四种日常语域（按摩广告/旅游手册/生物学讲义/课本引用）轮换讲价；关键抒情动作只用"眼角抽动"或"心里悄无声息地疼了一下"这种五到八字的极短句表述。
- 谨慎复用：死三次论、"有的人活着他已经死了"引用是名句改写，仿写时应换成另一句公共知识做同类操作。
- 不建议照搬：藏蓝和服 + 木屐白袜 + 红苹果脸的路鸣泽视觉签名是角色专属；番茄酱薯条属于下一场（s07）的引子。
