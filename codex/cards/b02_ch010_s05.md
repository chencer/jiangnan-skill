# b02_ch010_s05 · 送别的群像与无人愿去的目的地

## 基本信息

- **卷/章**：卷二 · 第三章《屠龙记》
- **锚点**：ch_010_第三章.txt L127-L145
- **场景类型**：B级 · 日常闲笔（会议收尾 + 群像送别 + 孤立收尾画面）
- **模式**：幽默模式（孤独藏在荒诞末尾）
- **人物**：昂热（被送走者）、阿卡杜拉所长（主送别者）、卡尔副所长、马突尔研究员、装备部众人

## 基本结构

会议结束，进入电梯厅。阿卡杜拉所长借用文化差异（点头/摇头）做最后一轮轻喜剧，随后以无法派遣技术人员随行为核心冲突——众人集体退后一步，各自用本民族的神明名义飞速亲吻完昂热后消失。最后只剩昂热一人，手里多了一本砖头厚的技术手册，电梯还没来。孤独感藏在空荡荡的走廊里，没有被点名。

## 技法清单

- **集体逃跑的仪式感**：装备部众人不是"借故离开"，而是用亲吻 + 宗教祝福 + 快进的仪式完成一次集体撤退。越庄重的形式，越露出逃避的本质。祝福词连发四条（真主、上帝、毗湿奴、阿弥陀佛），宗教多样性本身成了包袱。
- **文化差异的最后一用**：阿卡杜拉的点头/摇头问题在整章里反复出现，在这里做最后一次变形——昂热直接捧住他的脸不让他动，阿卡杜拉选择开口说话。动作解决了语言问题，但问题的本质（对方不愿配合）一点没变。
- **技术手册作实物锚**：抽象的"派人随行"被替换成一本砖头厚的手册砸进昂热掌心，重量感是具体的，责任转移是物理的。比任何台词都直接。
- **孤独不直说**：结尾没有内心独白，没有"他感到一阵空落"，只有空走廊、手里的手册、电梯还没来。三件事并置，不解释。
- **黑洞意象收尾**：日本分部的描述用"黑洞"作比，加上"血腥味随风飘来"做气味锚定。轻喜剧场面在这句话之后忽然有了重量，下一段的任务危险性已经悄悄埋进去了。

## 可复用度（B）

- **适用场景**：组织内部没人愿意接烫手山芋，领袖最终独自拿下；送别场面需要喜感和孤独感并存；需要用"没人"来衬托主角的处境，但不能煽情。
- **迁移公式**：
  1. 让集体逃跑有仪式感——越正式的告别方式，越能体现逃跑的心虚。
  2. 把抽象的责任转移变成一件实物递到主角手里（手册、账单、一把钥匙）。
  3. 收尾画面只放空间 + 主角 + 一件还没发生的小事（电梯没来、外卖还没到），不配内心独白。
  4. 如果需要预示后续危险，用别人的描述（"血腥味"、"没人知道里面有什么"）埋伏，不让主角开口分析。
