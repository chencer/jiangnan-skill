# b02_ch021_s07 · 神葬所苏醒对话

- 出处：第十四章 王的血祭
- 行号：L44-L50
- 场景类型：情感重场（战略布置 + 立场交底）

## 火力分配
- 信息密度：3.2级地震 / 神葬所苏醒 / 诺玛系统状态 / 联盟结束时间 全部在7行内交代完
- 情感潜流：主角回应"祖先们果真没有彻底死去"——把技术汇报接入禁忌历史，一句给整个家族背景开了口子
- 冷处理宏大：切断联盟这种级别的战略决定，用"已经不重要了"一笔带过

## 分镜序列
1. L44 樱走到背后，播报震度和状态（军事简报口气）
2. L45 主角回应，把地震语义翻转为"祖先想逃回人世"

4. L47 主角一句让樱去休息（关心 + 权力，藏在同一句里）
5. L48 樱继续报诺玛/辉月姬博弈
6. L49 主角战略拍板："已经不重要了，今夜过后……联盟就结束了"
7. L50 樱："明白。"——两字截断

## 技法清单
- **N02** 宏大场面轻描淡写：3.2级地震+神话级苏醒+断盟——全部用军情汇报的白描腔一笔带过
- **N06** 占有/情感宣告·塞进日常腔："祖先们果真没有彻底死去啊"——史诗级判断说得像家常闲聊
- **H03** 日常语汇收束宏大/神圣时刻："你去休息一下吧"接在神葬所苏醒之后
- **N01** 冷短句截断·不接情绪："明白。"两字收场
- **N08** 金句落点·接日常动作截断："今夜过后，联盟就结束了" → 立刻切到"一个小时之内阻止他……就好了"
- **T04** 双重视角空行并置（弱用）：本段和 s06、s08 之间靠场景切换默认并置

## 可复用度（A）
- 军情汇报体裁的通用模板：下属报硬指标 → 主角把技术信息翻译成历史意味 → 下属拉回执行面 → 主角关心 → 下属追加坏消息 → 主角拍板 → 下属"明白"两字截断
- 关键要害：所有"重大战略决定"都要用"已经不重要了""一个小时之内……就好了"这种轻飘飘的口气说出，反而更沉
- 危险点：不能让主角把话说满，宏大判断说完立刻切到具体指令（"一个小时之内"）
