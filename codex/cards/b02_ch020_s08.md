- 一句话：楚子航从一句「列宁号」引出推理链，锁定破冰船击穿海床，然后由肺螺的密集生育反推整个深渊生态链，最后一句话把胚胎坐标钉在众人正前方。

## 火力分配

- 推理链占 55%（列宁号缺席 → 沉入废墟内部 → 被贝类覆盖）
- 生态倒推占 30%（肺螺生育 → 磷虾 → 鱼群 → 龙蝰）
- 视觉与体感占 10%（贝类蠕动，头皮发麻）
- 收束句占 5%（一句话完成坐标定位）

## 分镜序列

1. L119 单词起手：楚子航只说三个字「列宁号」，把其他人从鸟居的震撼里拽回任务。
2. L121 推理铺开：破冰船参数、下沉动力学、船头击穿海床，逻辑一路推到「大部分仍然暴露在外面」。
3. L122-L124 质疑与合流：路明非拿钢罐类比 → 楚子航用「内外压平衡」驳回 → 恺撒接力得出「它被贝类覆盖了」。
4. L125-L127 短暂欣喜后再抬头：路明非叫大家别看照片了，视线转向观察窗上方——把整整一座建筑重新定义。
5. L128 视觉揭幕：黑色贝类被远距摄像机拉近，屏幕上「细小的铁锈在蠕动」。
6. L131 恺撒低声解释肺螺雌雄同体、腮腔孵化、每秒喷出成千上万小肺螺。
7. L133 楚子航把整条食物链讲完：肺螺 → 磷虾 → 鱼群 → 龙蝰。
8. L135 楚子航一口气收尾：「龙的胚胎就在我们的正前方……我们找到它了。」

## 技法清单

- N04 伪装破绽·读者先于角色知情·延迟引爆——那座「像山一样的废墟」其实一直在视野里，只是等楚子航一句话把它翻译成列宁号。
- N02 宏大场面轻描淡写——「它在废墟内部！」用短促陈述句处理石破天惊的发现。
- V01 感官链·视觉主轴——远距摄像机 → 铁锈 → 蠕动 → 头皮发麻，视觉主轴带出体感。
- M01 比喻·降格（日常器物代宏大）——把在深海远古废墟表面蠕动的贝类，喻为「蛆虫在大象尸骨上钻进钻出」，用最生活的恶心画面拆掉神圣感。
- M02 比喻·死亡史前置——大象尸骨、蛆虫，喻体自带腐烂气味。
- T03 时间多段伸缩——推理段落把二十年下沉压成几句物理分析，肺螺段落又把「每秒钟」拉成慢镜。
- N01 冷短句截断·不接情绪——「我们找到它了。」六字关门，不铺情绪。
- N03 神性/巨物降格·日常语汇化——恺撒用「砰啪一声把硫磺炸弹发射出去，我们掉头就回家」处理屠古龙。
- H01 想/说落差——路明非心里怕得肝颤，嘴上却是「打到哪里算哪里」。

## 可复用度（A）

- 强复用：单词起手 + 三段推理 + 一句坐标（可套用于任何「破案式」场景收束）。
- 强复用：把恶心生物写成一整条食物链——从最小的生物一路推到顶端捕食者，再把捕食者身份接到主角这一次任务的目标上。
- 强复用：「别看照片了，看上面」这种把视线从二维拉回三维的调度句式。
- 中复用：肺螺雌雄同体、腮腔孵化这一类硬知识 + 感官恶心的组合，剂量要控。
- 慎用：「大象尸骨上的蛆虫」这类喻体太强，一章一次。
