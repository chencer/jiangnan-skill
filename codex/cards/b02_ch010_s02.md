# b02_ch010_s02 · 危机披露的情报会议

## 基本信息

- **卷/章**：卷二 · 第三章《屠龙记》
- **锚点**：ch_010_第三章.txt L35-L56
- **场景类型**：B级 · 高压冲突（危机情报披露 + 荒诞插科 + 领袖被动承压）
- **模式**：幽默模式
- **人物**：昂热（被召回的领袖）、卡尔副所长（汇报者）、装备部众人（旁观者兼插科打诨者）

## 基本结构

会议开场由昂热的自嘲台词定调，随后通过一段声音素材做信息投递的缓冲。卡尔副所长的信息按"现象→研判→风险等级"三层叠加，每一层加压之后都跟着一句荒诞插嘴把气氛泄一口气。昂热的反应始终是提问，不表态，用问题的密度代替直接的情绪。

## 技法清单

- **声音媒介入场**：危机不靠台词直接说出，先用音频播放做感官准备。听者从"普通录音"到"察觉异响"，认知跟着升温，省去了大段铺垫。
- **身体动作代替情绪**：领袖在关键时刻的反应不是台词，而是握紧折刀、手背青筋——把内心的震动压进一个微小的肌肉动作里，不让它溢出。
- **三层加压结构**：信息一层一层往深推（声音→心跳→龙类→胚胎→孵化倒计时），每层之间插一句轻飘飘的补充或者反问，像吸气再呼气，让读者跟住节奏。
- **旁观者插嘴做阀门**：关键的沉重结论刚落，立刻有人拿"早期肺癌"做类比，气氛瞬间被别针戳破。目的不是搞笑，是阻止场面变得过于沉重而失去张力。
- **问题驱动的领袖形象**：昂热不主动发表情绪，用连续提问（孵化时间？目标级别？沉船联系？）替代表态，问题本身显示出他已经在运转。

## 可复用度（B）

- **适用场景**：高层被叫回来处理坏消息；情报会议中，危机信息需要逐层加压而不能一次打完；领袖角色需要显得"在思考"而不是"在反应"。
- **迁移公式**：
  1. 用感官媒介（声音/气味/图像）做信息的前置缓冲，让读者比角色早半步感知到威胁。
  2. 关键情绪动作用手、眼、脊背等微小部位承载，不用心理描写。
  3. 每一轮加压后安排一句调子明显不对的台词，防止气氛凝固。
  4. 汇报者用专名和数字锚定现实（日本海沟、八公里、心跳频率），让未知威胁有了具体重量。
