# b02_ch021_s12 遗书文笔评价与楚子航短版遗书

## 基本信息
- 位置：ch_021 第十四章 王的血祭
- 行号：L91-L102
- 类型：日常闲笔
- 火力级别：A
- 概述：恺撒抒情独白刚落地，楚子航立刻用"畅销书作家"的评语把它降格为文学作品；随后播放自己那段只讲给继父的极简遗书，两份遗书形成对照，源稚生在须弥座顶端旁听。

## 火力分配
- 情感浓度：中偏高（藏在幽默腔里）
- 密度覆盖：对话六到八轮
- 命中层：a 情绪落差 / b 幽默技巧 / c 结构对照
- A 级理由：把顶级情感重场"续"成一段书评闲谈，落差极大；用楚子航"没什么想跟妈妈说的"再抛出第二枚冷弹；结尾让源稚生一言不发地叼着烟旁听，把两条时空焊在一起。

## 分镜序列
1. L91-L92 楚子航忽然评价恺撒私下出的畅销书，把遗书当作《Dragon Raja》第三本的宣传物料。
2. L93-L94 承认文笔好但吐槽中文用词，恺撒反将一军："追中国女孩不学中文怎么行"。
3. L95-L96 楚子航播放自己那段"给继父"的短遗书，只有几句：请勿追查/请照顾妈妈/劝她再生一个。
4. L97-L99 恺撒与路明非追问为何不留给妈妈，楚子航冷静回答"她没有血统，不会明白"。
5. L100 楚子航反手把话头递给路明非："你要不要录音？"
6. L101 路明非说想过要录，但没想明白录给谁听——用抓头收束。
7. L102 空行焊接：源稚生叼着烟卷靠在栏杆上，默默地听着海底深处的对话。

## 技法清单
- H01 想/说落差：楚子航听完最重的情话，开口第一句是《纽约时报》畅销书榜。
- N03 神性/巨物降格·日常语汇化：把"死前告白"评为文学作品，用书评腔化解神圣感。
- H03 日常语汇收束宏大/神圣时刻：以"畅销榜第三名/计划推出第三本"结算遗书。
- N06 占有/情感宣告·塞进日常腔：楚子航的遗书全是事务性交代（勿追查、再生一个）。
- T04 双重视角空行并置（禁过渡句）：海底三人对话与须弥座顶端源稚生旁听，空行连接，无过渡句。
- N08 金句落点·接日常动作截断：路明非"没想明白录给谁听"后接一个抓头动作，不升华。
- H02 越荒诞越平静口吻：谈死用最平的语气，"没有人是不可替代的，也没什么好难过"。

## 可复用度（A）
- 高复用：重场之后紧跟一句"文学评论式"降格；用"畅销书排名"结算情感事件；结尾把话筒递给最不想接的人。
- 谨慎复用：跨语言吐槽（"中文用词有时候不准"）需要人物本身有外籍设定支撑。
- 不建议照搬：具体书名 Dragon Raja 是元梗，属于作者本人的自指，模仿时要换成其他自造 IP。
