# b01_ch004_s04 黑天鹅港天台的誓约

## 基本信息
- 场景类型：情感重场（弧形空间漫游 → 制高点俯瞰 → 礼物交换 → 死亡观交易 → 龙族血统显现与逃亡誓言）
- 在场人物原型：零号（无所不能的少年，龙族血统初显）、雷娜塔（被囚养的女孩，抱着玩具狗佐罗）
- 核心目的（一句话）：把一场儿童间的送花、耍宝、讨吻，写成一份牵涉死亡与自由的血契。

## 火力分配
- 重墨对象：
  1. 教堂穹顶推窗那一瞬的极地景观（冰壳、幽蓝水道、地平线下的太阳、酡红一抹）——用整段风景压出"离极点近，离人间远"的孤悬感
  2. 白铁盒里的枯罂粟：由礼物 → 学名 Papaver radicatum → "每一次死亡都为归来"的箴言，三级升维
  3. 结尾的凝视：金色水波、数千年的目光——一次性把男孩身份从人推到非人
- 简笔带过：
  - 通道穿行只用"虚掩的门""沉默的警报器""锈铁梯"三个意象拼贴，一句"今夜港口的每个角落都对他们开放"收拢
  - "小败狗 / 小海豹"的绰号绕圈只用四五句对话回合完成，不注释潜台词
  - 档案室三道锁、二号柜最下抽屉——一句话交代零号的越权能力
- 支线位置：烟囱铁闸放在情绪最冷时（"离人类世界那么远"）——一股炭火把心里的寒气顶回去，作为物理支点承接下文的坐下、依偎、哼歌

## 分镜序列
1. 中景跟拍：手拉手在弧形通道里疯跑，笑声起飞（动感）
2. 仰拍推入：钻进教堂，踩十字架，肩上骑姿推开彩窗
3. 大远景切入：冰壳、冰峡、水道、酡红的天——**时间伸缩点①** 视野一开，心理年龄瞬间被拉到成年
4. 中景：天台十字架，红军战士名字（历史压舱石一句带过）
5. 特写台词×2：距北极点 453 公里 / 距莫斯科 3781 公里（用精确数字锚定荒凉）
6. 近景：铁闸掀开、烟气升腾、并肩坐下、哼不知名的歌
7. 特写交接：白铁盒 → 枯罂粟 → 学名 → "每一次死亡都会为了归来"（**时间伸缩点②** 一句古文献口吻的箴言把童声压出千年感）
8. 对话拉锯：全名、档案、爸妈胡子的酒气、"汪！汪！"耍宝、小败狗 / 小海豹
9. 顿点：雷娜塔说"让我死"——空气冷下来
10. 反打特写：零号磨牙吮血的话语，声音却温柔（**时间伸缩点③** 一句话里塞入两种年龄）
11. 收束长镜：按掌为誓，"直到死亡的尽头"
12. 定格：眼底金色水波，凝视长达数千年——**时间伸缩点④** 一个孩子的眼神里落进整个种族的年岁
13. 三字收尾：我愿意。（对话截断，不做叙述总结）

## 技法清单
- 比喻：
  - "像在飞翔"——只此一次直白比喻，用在疯跑，之后不再出现
  - "眼前仿佛世界尽头，美得让人觉得那么孤单"——景 → 情，只一句，不解释
  - "他捏着她的心思就像捏着属于自己的东西"——只写捏的动作，不写"她被吃定了"
- 感官传导链：
  - 视觉（冰壳幽蓝）→ 触觉（寒风扑面）→ 想哭的冲动 → 短句"美得让人觉得那么孤单"
  - 触觉（炭气涌上）→ 心理（寒气退散）→ 姿态（挨着坐）→ 听觉（哼歌）——四拍完成从冷到暖
  - 触觉（掌心相按）→ 视觉（金色水波）→ 时间感（数千年）——从人的接触滑进非人的凝视
- 对话：
  - "你叫我什么？"愣住式反问打开身份缺口
  - "让我死"——五个字的极端愿望，不铺垫、不修饰
  - "挡你路的……才该死"——三段停顿把儿童语气切成成年宣告
  - "我愿意"——三字回礼，直接落地，无叙述尾音
- 金句（可提炼）：
  - "世界上永远有一种生命，它的每一次死亡都会为了归来。"
  - "我以前哭得太多，已经没有哭的能力了。"
  - "他说着磨牙吮血的话，可声音那么好听那么温柔。"
  - "他的凝视漫长悠远，长达数千年。"

## 可复用度
A｜双人天台戏的高配模板：**弧形漫游→制高点极景→暖源支点→礼物升维→死亡观交易→血统闪现→誓言短促收束**，七步之内把普通儿童场景抬到史诗契约级；尤其"炭火烟囱"这类反冷源意象和"数千年凝视"的一次性血统闪光，值得拆出来单用。
