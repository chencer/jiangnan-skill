# 技法标签规范词汇表 v1.0

> 来源：从 40 张 A 级卡片签名归纳，共 30 个规范标签。
> 用途：a_level_registry.md 签名统一使用本词汇；相同组合碰撞时降 B。
> 更新：2026-07-10（Round 3 P1）

---

## 感官组 V

| 标签 | 全称 | 定义要点 |
|---|---|---|
| V01 | 感官链·视觉主轴 | 感官推进以视觉为主轴，其他感官辅助，不完全走完五感 |
| V02 | 感官全链·完整序列 | 视→触→听→嗅→体感→身体反应全链，最终落在生理变化 |
| V03 | 感官三段跳·幻觉认证 | 颅内感受→自我怀疑→物理同频验证→行动接续 |
| V04 | 感官链·时空焊接 | 往昔感官记忆与当下感官刺激交叉剪辑，不标注时间切换 |

## 比喻组 M

| 标签 | 全称 | 定义要点 |
|---|---|---|
| M01 | 比喻·降格（日常器物代宏大） | 用日常器物/行为作为喻体，把神圣/宏大/危险日常化 |
| M02 | 比喻·死亡史/负面史前置 | 喻体本身带死亡或血腥背景，喻体的历史进入比喻 |
| M03 | 比喻·物种/神性升格 | 把人物拔高到动物、神、古生物层级，完成身份越界 |
| M04 | 比喻·反差命名（以X定义X） | 用同一类别中最极端的成员来定义这个类别（魔鬼定义魔鬼）|

## 幽默组 H

| 标签 | 全称 | 定义要点 |
|---|---|---|
| H01 | 想/说落差 | 内心想法（口语/吐槽）与说出来的话（意外史诗或日常）之间的落差 |
| H02 | 越荒诞越平静口吻 | 场面越离谱，叙述语气越冷静，不用感叹号 |
| H03 | 日常语汇收束宏大/神圣时刻 | 商业语、报菜名语、广告语用于神圣/史诗时刻，将其日常化 |
| H09 | 自我辩护·递进被反噬 | 三段式自我辩护，每层递进，最终结果反证自己 |
| H10 | 翻译腔/文化误读·长线梗 | 跨文化误读或机翻腔调，埋梗跨越多个场景才引爆 |

## 道具/触觉组 P

| 标签 | 全称 | 定义要点 |
|---|---|---|
| P01 | 道具揭幕反转 | 一件物件出场后改变观众/主角对当前局势的判断 |
| P02 | 道具无言情感承载 | 物件动作替代情感表达，不加任何情绪词 |
| P03 | 触觉循环封印 | 触觉感受作为场景开场锚点，收束时以触觉完成封印 |

## 结构组 T

| 标签 | 全称 | 定义要点 |
|---|---|---|
| T01 | 章首末同句闭环 | 章节开头一段无主语画面/台词，结尾一字不差复现 |
| T02 | 倒计时锁定结构 | 明确量化递减（距离/时间/血量），制造压迫感 |
| T03 | 时间多段伸缩 | 同一段落内极度压缩与极度拉伸交替出现 |
| T04 | 双重视角空行并置（禁过渡句） | 两段互相不知情/立场相反的叙事并列，中间空行，不写连接 |

## 叙事调性组 N

| 标签 | 全称 | 定义要点 |
|---|---|---|
| N01 | 冷短句截断·不接情绪 | 金句或重场结束后，紧跟一个具体动作/短句，不升华 |
| N02 | 宏大场面轻描淡写 | 末日、历史、战争等宏观事件用白描/档案语气一笔带过 |
| N03 | 神性/巨物降格·日常语汇化 | 神/龙/暴力用日常副词（随手、有点）或吐槽腔处理 |
| N04 | 伪装破绽·读者先于角色知情·延迟引爆 | 读者先看见破绽，等待角色撞上，制造提前几秒的冷意 |
| N05 | 身体行为替代情感表述 | 不写"他很难过"，写他做了什么身体动作 |
| N06 | 占有/情感宣告·塞进日常腔 | 最深的爱/占有用商业合约、条件句、贬语说出来（反抒情的抒情）|
| N07 | 归因反转·降格自救 | 主角自救动作被降格为无效，救赎功劳归给外部 |
| N08 | 金句落点·接日常动作截断 | 金句落地后接一个完全日常的动作/追问，不让情绪停留 |

## 意象组 I

| 标签 | 全称 | 定义要点 |
|---|---|---|
| I01 | 末日/巨物意象叠合日常时刻 | 末世级意象（蘑菇云/死龙/神迹）与日常场景同帧并置 |
| I02 | 射灯揭幕·庞然大物首现 | 照明突然亮起，主角俯视/仰视揭幕，庞然大物首次视觉冲击 |
