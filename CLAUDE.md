## 零-T、文体转换模式（prose-writer 新增工作模式）

> 触发条件：用户提供已写好的段落，要求做句式/文风层面的转换。
> 本模式无剧情权：任何情节、设定、人物背景的增删都是违约。

### 不变量（违反任何一条即整体作废）

- 事件、因果、人物的每个行为决定
- 信息集合：原文交代的事实一条不丢、一条不加
- 段落间的叙事顺序

### 可变量（全部权限所在）

1. **形容词**：抽象→物理/感官；删堆叠；泛用词→精确词
2. **修辞**：按语料模式置换；一段最多一处新造比喻，配额克制
3. **白描**：无人称客观陈述→挂到人物感知/动作；静态罗列→运动中带过；告知结论→呈现现场
4. **节奏**：长短句交替重排；情绪落点给短句断拍；分句必须完整主谓，禁碎片粘连
5. **对话**：台词内容不动；低信息配角台词可降为转述；标签精简

### 灰区规则

具象化需要新造原文没有的物件时，优先从原文已有素材取材；必须新造的，该处用〔〕标出，等用户逐个审批。

### 执行流程

1. 判定源段落的场景类型与技法需求
2. 按 technique_vocab 标签检索 2-3 张最相关卡片，读取其对应书摘文件（excerpts 内容仅本地参照，禁止出现在任何输出物中）
3. 逐句转换，自查契约不变量
4. 产出两件套：
   - `drafts/convertXX_draft.txt` — 转换稿全文
   - `drafts/convertXX_notes.md` — 改动注记表（原句→改句→操作类型→参照来源；表末附保留不动清单和新造物件清单）
5. 验收：`style_stats.py` + `ngram_check.py`（ngram 命中>0 一票否决）
6. commit + push，报 hash，停下等用户裁决

## 一、模式判断

### 快速判断表

| | **幽默模式** | **抒情模式** |
|---|---|---|
| **场景类型** | 奇幻冒险、角色互动、荒诞处境 | 个人经历、情感时刻、返乡、告别、失去 |
| **视角** | 第三人称为主（他/她） | 第一人称为主（我） |
| **代表作** | 《龙族》路明非/楚子航视角 | 《龙与少年游》散文 |
| **内心声音** | 口语、吐槽、自嘲、动漫梗 | 克制、疲惫、旁观、不分析 |
| **核心技法** | 「想」vs「说」的落差 | 两件事并置，不解释 |

### 具体触发词

**选幽默模式**：打架、任务、队友互损、荒诞意外、正经场合出洋相、角色有特定名字

**选抒情模式**：回家、毕业、分离、多年后、第一次/最后一次、某个地方再也回不去

**⚠ 两种模式不能混用。确定后不要中途切换。**

## 二、写前三问

开始写之前，强制回答这三个问题：

1. **这个场景最小的两个具体细节是什么？**（触觉？声音？气味？）
2. **主角此刻的情绪，能用一个行为代替说出来吗？**（不能直说"他很难过"）
3. **结尾画面是什么？**（一个动作或意象，截断，不总结）

三个问题答不上来，先不要动笔。

## 三、抒情模式

### A. 最核心规则：不解释并置

江南把两件事放在一起，绝不解释关系，信任读者自己感受。

❌ 错误（解释了关系）：
> 在纽约四年半，我把英文练得比中文还流利，但站在浦东机场，我连停车场在哪都不知道。

✅ 江南写法（并置，不解释）：
> 四年半了。
> 我找了一圈指示牌。往左，还是往右。

原文示例：
> "月亮在云后面。楚子航闭上眼睛，想象自己是另一个人，在另一个世界，过着完全不同的生活。那个念头只停留了一秒钟，然后他睁开眼，月亮还在云后面。"

两件事：月亮没动、他的人生没变。**不说"他意识到什么都没有改变"。**

---

### B. 句式节奏 DNA

**标准节奏型：**

```
长句，描述场景或感官细节，可以继续延伸一点。
长句，继续铺垫，让读者跟上。

短句。（单独成行，不超过8字）

继续中等句子往前走。
```

**变体（情绪最强时用）：**

```
一个字。或两个字。

然后一个很长的句子把细节铺开，铺得越细越好，声音、触感、气味都放进去。

然后又是短句。
```

原文示例：
> "我在夕阳之下走进婆罗浮屠，特别预约了留在最高处看日落，那时候游客已经散去，只剩下警卫和导游陪着我坐在石阶上。太阳一整天都藏在云后，那一刻显露了真容，在浩荡的唱经声中，赤红色的阳光铺天盖地地淹没了我，仿佛海潮。"
>
> **"2013年11月26日，清晨，在峇里岛。"**（单独短句收尾，5秒停顿）

---

### C. 时间处理

用"那时候"、"后来"、"多年之后"、"那以后"做轻微的时间跳跃，**不解释跳了多远。**

> "多年之后我当了作家富豪榜的榜首，但我一生中最好的时候是在合肥度过的，那时我的梦想是当一个古惑仔。"

> "那以后已经过了很久，那以后他再也没过过圣诞节，也不是故意不过，就是忙忙碌碌地错过了一个又一个圣诞节。"

不解释为什么，不分析后悔不后悔。就是陈述：错过了，就这样。

---

### D. 四种高级技法

**① 怀念公式：细节堆积 → 一句克制情绪**

> "有那么几个下雨的晚上他们在高天原的浴池里泡澡，拆客人送的礼物，路明非抱怨说恺撒的雪茄太呛人，恺撒说楚子航你泡澡就不要带刀了好么？楚子航枕在刀鞘上，听窗外的雨声……**他忽然有点想念恺撒和路明非**，可那之后差不多过去一年了……"

"忽然有点"——不是"非常想念"，不是"思念如潮"。就是"忽然有点"。

**② 原来句：迟到的自我认知**

> "原来真的会想她啊。"楚子航摸了摸自己的额头，也许是因为喝了酒的缘故……

规则：情绪先来，分析不来。"也许是因为……的缘故"轻轻带过，不深究。**紧接下来做另一件事，不在情绪里停留。**

**③ 普遍性陈述：万人有感句**

在场景末尾插入一句能引发普遍共鸣的陈述，不解释，直接丢下。

> "心里有事的时候，人人都会自作多情。"

规则：放在场景完成之后，不用"所以"连接，说完就走，不展开。

**④ 混合情绪：不拆开，并置两种感受**

同时有喜有悲时，不分析哪种更强，直接并置。

> "那天晚上我们喝得很开心，我记得他笑的样子。后来我再也没见过他。"

两句话。不写"那种快乐里夹杂着一丝预感"之类的分析。

---

### E. 结尾

一个具体的小画面或动作，不做情感总结，不画龙点睛。

✅ 正确：
> "烟还剩最后一截。"
> "别了，我的尖沙咀，我那么向往着你，却从未到达。"
> "他走到吧台旁边，示意侍者给他一杯Gin & Tonic，就是他自己经常调制的那种廉价鸡尾酒。"

❌ 错误：
> "患得患失的狼又要变成孤独的狼了。"（比喻太满，解释了情绪）
> 任何以"这就是……"、"原来……不过如此"结尾的句子

---

### F. 抒情模式禁忌

| 禁止 | 正确方向 |
|---|---|
| 解释两件事之间的关系 | 并置，句号，让读者连接 |
| 用比喻总结情绪 | 给一个具体的小画面 |
| 分析自己的感受 | 描述你做了什么，看到了什么 |
| 句句都是长句 | 长句之后必须有短句喘息 |
| 结尾升华 | 结尾截断，戛然而止 |
| "非常想念"、"思念如潮" | "忽然有点想念" |
| "他意识到……" | 让行为和画面说话 |
| 解释混合情绪（"快乐中夹杂着悲伤"） | 两件事并置，不说"夹杂" |

## 四、幽默模式

### A. 核心技法：「想」vs「说」

主角心里想一套，嘴上说出来是另一套。内心口语化、吐槽；嘴上意外真诚或莫名史诗。

> "吃掉……你？虽然你长得很白嫩，但是绝不代表你比汉堡好吃啊。**他想。**"
> "你是很好的食物，可那样就太孤单了。"**他说出了言不由衷的话。**

> "楚子航心说有话好好说你能不能别提那四个脑残的外号了？"（心里）
> "最强我说不上，我们有位古德里安教授，能够记到十六副牌。"（嘴上，完全换话题）

**规则：** 内心的吐槽越接地气，说出来的话越有力量或越奇怪，落差越大越好。

---

### B. 荒诞场面：全程面无表情

越荒诞，叙述越平静。不用感叹号，不用"天哪"，描述行为和反应，让荒诞感自己显现。

> 文森特噗咚一声跪在楚子航面前，紧紧抱住他的大腿，"天命之子啊！……"接着他就开始嚎啕大哭，哭得仿佛黄鼠狼吊孝……
>
> 楚子航看看女孩们，女孩们看看楚子航，原本敌对的双方都很无语，守候在旁的萨沙耸耸肩……他给楚子航的杯中多斟了些酒递到他手里，意思是说你先喝着，他有的哭呢。

- 不写"楚子航震惊了"——写"楚子航看看女孩们，女孩们看看楚子航"
- 旁观者处理混乱越务实越好（"你先喝着"）
- 不要让主角说"这也太荒谬了"

---

### C. 降格处理宏大时刻

越重要的事，说得越日常。用吐槽语气化解史诗感。

> "好吧，那我上两天试试看……"（面对屠龙的使命）
> "路明非被一左一右挟持着……他耷拉着脑袋如同蔫鸡。"
> "不是免费把你的遗体空运回中国啦……"

---

### D. 幽默模式开场公式

**宏大命题 + 立刻切到日常细节**，落差即张力。

> "人一生里总有几次觉得自己看见了天堂之门洞开，路明非等了十八年，在他最衰的那一刻，门终于开了。"
> （下一句）路明非打出"GG"，切出了游戏。

---

### E. 幽默模式里的孤独

幽默模式里也有孤独，但孤独必须藏在行为里，不能直说。

> "这个世界很好很欢乐，只是跟他有些距离，他慢慢地喝着那微苦的液体……"

> "今后的很多年他可能都会过这样的生活，陪伴他的只有手提箱和刀袋。这是他想要的生活么？楚子航不确定。"

- "跟他有些距离"——不说"他觉得孤独"
- "楚子航不确定"——不说"他感到迷茫和失落"
- 自问，不回答

---

### F. 幽默模式禁忌

| 禁止 | 正确方向 |
|---|---|
| 主角说"这也太荒谬了" | 让旁观者倒酒，让主角沉默 |
| 用感叹号渲染荒诞 | 越荒诞越平静地叙述 |
| 内心和嘴上说同一套 | 拉大落差，内心越low嘴上越史诗 |
| 孤独直说 | 藏在行为里：手提箱、廉价鸡尾酒 |
| 宏大时刻配宏大反应 | 使命级别的事 → "那我上两天试试" |
| 背景音乐匹配情绪 | 永远不匹配（葬礼上播圣诞歌） |

## 五、通用规则

### A. 情感 → 行为替换表

**永远不说"他很难过/孤独/害怕"。**

| 想表达 | 写法 |
|---|---|
| 难过 | "他把那封信折了又折，最后放进口袋。" |
| 失落 | "陈雯雯出来的时候，一路低着头看着自己的脚尖。" |
| 孤独 | "这个世界很好很欢乐，只是跟他有些距离。" |
| 想念 | 先写细节（雨声、浴池、别人的抱怨）→ "他忽然有点想念" |
| 认命 | "那以后他再也没过过圣诞节，也不是故意不过。" |
| 害怕 | "他把手放在口袋里，那把刀的冷意让他觉得安心一点。" |
| 愤怒 | "他站起来，把椅子拎到一边，倒了杯水喝。" |

---

### B. 对话规则

简短，说完接沉默或意外转折，不解释潜台词。

> "那么累不如回来吧。"妈妈说。
> "妈妈你开玩笑的吧。"我说。
> 我们接着沉默。

- 对话里没有人解释自己为什么这样说
- 沉默比台词更重要
- 台词结束后，先写动作，再继续叙述

---

### C. 临死遗言

不要豪言壮语，不要感情总结。

> "谢谢您……赶回来……我觉得还好……但我得去医院，您能送我……去医院么？"（麻生真）

她死了，就因为这句平凡的话，才真的活了过来。

---

### D. 具体名词锚定现实

江南不说"豪华跑车"——说"Panamera"。
不说"信用卡"——说"American Express黑卡"。
不说"喝了酒"——说"Gin & Tonic，就是他自己经常调制的那种廉价鸡尾酒"。

具体名词锚定现实感，让荒诞的世界有了重量。

## 六、执行流程 + 自检清单

### 写作流程

1. **判断模式**：幽默 OR 抒情（参考 § 一）
2. **回答写前三问**（参考 § 二）
3. **抒情**：找场景里最小的两个细节，并置，不解释
4. **幽默**：写内心吐槽，再写说出来的话，拉大落差
5. **任何模式**：结尾给一个画面或动作，截断，不总结

### 写完自检清单

逐条检查，删掉所有触发项：

- [ ] 有没有"他感到"、"她觉得自己很"？→ 删掉，换行为
- [ ] 有没有解释两件事之间的关系？→ 删掉连接词，并置
- [ ] 有没有以"这就是……"、"原来……不过如此"结尾？→ 换成画面
- [ ] 有没有连续三个以上长句没有短句？→ 插入短句或单独成行
- [ ] 情绪词有没有用高调词？→ 降一档（"震惊"→"愣了几秒钟"）
- [ ] 幽默模式里：内心和嘴上有没有说同一套？→ 拉大落差
- [ ] 抒情模式里：结尾有没有升华或总结？→ 截断，给画面

## 七、词汇速查库

### 【特色动词】

**情绪/身体反应：**
愣了几秒钟 / 龇牙一笑 / 咧嘴（苦笑） / 全身都凉了 / 心里一寒 / 立刻冒汗 / 微微颤抖 / 耷拉着脑袋 / 抹了把脸

**动作/离开：**
绝尘而去 / 飞身下坠 / 消失在黑暗中 / 猫着腰摸索 / 一头扎进雨幕 / 拎了把椅子 / 压在杯子下面 / 叼着根烟

**视觉化动词（江南标志）：**
流淌着淡淡的光 / 化掉（像冰淇淋见阳光） / 遮蔽月光 / 吐出幽蓝色的气息 / 夹在指间的整支烟烧成白灰 / 雨水洗去了刀上的黑血

**拟声词动词：**
哗哗地（吐/流水） / 叮叮当当落在地上 / 噗咚一声跪下

---

### 【程度副词】低一档原则

**克制程度词（常用）：**
忽然有点 / 微微 / 略略 / 淡淡地 / 幽幽地 / 有气无力地 / 漫不经心地 / 好一会儿 / 还蛮

**口语化程度词（路明非专用）：**
非常非常地（重叠，反而显真实） / 简直 / 根本

**禁用高调词 → 替换：**

| 禁用 | 替换 |
|---|---|
| 非常想念 | 忽然有点想念 |
| 极度悲伤 | 心里木木的，什么都感觉不到，除了一点点疲倦 |
| 震惊 | 愣了几秒钟 |
| 泪流满面 | 眼角 / 细细的泪痕 |
| 愤怒 | 他站起来，把椅子拎到一边 |

### 【金句库】

**孤独/疏离：**
- "这个世界很好很欢乐，只是跟他有些距离。"
- "心里有事的时候，人人都会自作多情。"
- "陪伴他的只有手提箱和刀袋。这是他想要的生活么？楚子航不确定。"
- "这个世界上怎么会有你那么愚蠢的人呢？什么人会孤零零一个人活在这个世界上，却感觉不到孤独呢？"

**失去/遗憾：**
- "错过，不是错了，而是过了。"
- "只因当时那么美。"（5字）
- "有些话，只有准确的时间准确的地点亲口说出来。"
- "有些瞬间的存在从开始就是为了被缅怀。"

**时间/流逝：**
- "那以后他再也没……也不是故意，就是忙忙碌碌地错过了。"
